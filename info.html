Progress through skills development


Different shaped badges for Skill Areas:


White Belt

These are entry level skills and mainly focus on learning the tools and health and safety aspects of a specific skill.

Examples of these types of skills for a Raspberry Pi white belt badge would be:

  • Draw a diagram explaining the function of the components on a Raspberry Pi.
  • Learn how to connect up a Raspberry Pi.
  • Use a Raspberry Pi to switch on a LED.

Science

This will cover skills in the Science field examples of these skills areas are:

  • Experimentation
  • Sound Waves
  • Light
  • Atoms
  • Astronomy
  • Chemistry

 

Yellow Belt

Once someone has completed three core white belt level skills they can move on to yellow belt projects which will build on the knowledge and skills they have already gained.

Examples of these types of skills for a CSS Yellow belt badge would be:

  • How to make shapes using CSS.
  • How to create an external style sheet and use in all your web pages.
  • How to position items using CSS.


Technology

This will cover skills in the technology field examples of these skills area are:

  • Arduino
  • Raspberry Pi
  • CSS
  • HTML
  • 3d Printing
  • Robotics

Orange Belt

Once someone has completed three core yellow belt level skills they can move on to orange belt projects which will build on the knowledge and skills they have already gained.

Examples of these types of skills for a Arduino Orange belt badge would be:

  • Controlling a Servo Motor.
  • Creating a Oscillating Synthesizer.
  • Working with am Arduino shield.

Engineering

This will cover skills in the Engineering field examples of these skills areas are:

  • Electronics
  • Soldering
  • Gears
  • Motors
  • Structures
  • Mechanics

 

Blue Belt

Once someone has completed three core orange belt level skills they can move on to blue belt projects which will build on the knowledge and skills they have already gained.

Examples of these types of skills for a WordPress blue belt badge would be:

  • How to change and configure the theme on WordPress.
  • How to create a Photo slider using a Plugin.
  • How to back up your WordPress site.

Art

This will cover skills in the Art field examples of these skills areas are:

  • Colour Theory
  • Graphics Design
  • Photography
  • Drawing
  • Painting
  • Sculpture

 

Brown Belt

Once someone has completed three core orange belt level skills they can move on to blue belt projects which will build on the knowledge and skills they have already gained.

Examples of these types of skills for a Robotic brown belt badge would be:

  • Build a Sumo Robot.
  • Program a Maze solving robot.
  • Use Open Computer Vision to follow a face.

Maths

This will cover skills in the mathematics field examples of these skills are:

  • Algebra
  • Trigonometry
  • Geometry
  • Calculus
  • Arithmetic
  • Budgeting

Black Belt

Once someone has completed three core brown belt level skills they can move on to black belt projects. These will be the most technically challenging level of skills and stretch the children's knowledge and utilise all the skills they have learned so far.

Examples of these types of skills for a 3d Printing black belt badge would be:

  • Printing with multiple types of filament in the same print.
  • Designing a print with multiple moving parts
  • Advanced finishing techniques

Projects Overview

Projects will be multifaceted and include skills for more than one core skills group. This is to allow for the practical application of the skills and also to demonstrate that skills are not used in isolation. This will line up extremely well the the Scottish Government's Curriculum of excellence.
